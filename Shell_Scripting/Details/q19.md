## ✅ Coding Question #19 — Rotate Logs When They Exceed a Certain Size

### Question

Write a **shell script** that:

* Checks the **size of a log file**
* If the log file size is **greater than 100MB**
* **Archive (rotate) the log**
* Start a **new empty log file**

Example log file:

```text
/var/log/app.log
```

---

## 💡 What the interviewer is testing

They want to see if you know:

* `du` or `stat`
* conditional checks
* file renaming
* log rotation concepts

Log rotation is **very important in production systems** because logs can fill the disk.

---

## 🧪 Script Solution

```bash
#!/bin/bash

logfile="/var/log/app.log"
max_size=100000   # in KB

size=$(du -k $logfile | cut -f1)

if [ $size -gt $max_size ]
then
    timestamp=$(date +"%Y%m%d_%H%M%S")
    mv $logfile ${logfile}_$timestamp
    touch $logfile
    echo "Log rotated"
else
    echo "Log size normal"
fi
```

---

## 🔎 Explanation

| Command   | Purpose                      |
| --------- | ---------------------------- |
| `du -k`   | Get file size in KB          |
| `cut -f1` | Extract size value           |
| `mv`      | Rename the log file          |
| `touch`   | Create a new empty log file  |
| `date`    | Add timestamp to rotated log |

---

## Example

Before rotation:

```
app.log
```

After rotation:

```
app.log_20260314_221000
app.log
```

Old log is archived, new log starts fresh.

## ⭐ Follow-up Questions Interviewers Ask

### ❓ What tool normally handles log rotation in Linux?

Answer:

```
logrotate
```

Example config file:

```
/etc/logrotate.conf
```

---

### ❓ Rotate logs daily

Using **logrotate config**:

```
daily
rotate 7
compress
```

Meaning:

* rotate daily
* keep last 7 logs
* compress old logs

---

### ❓ Compress rotated logs

```bash
gzip app.log_20260314
```

## 🧠 Real SRE Insight

In production, logs grow quickly due to:

* application logs
* container logs
* audit logs
* debug logs

So systems use **log rotation policies** to prevent **disk exhaustion**.

Tools used:

* `logrotate`
* Kubernetes log rotation
* Docker log driver limits
* Cloud logging retention
