## ✅ Coding Question #3 — CPU Monitoring Script (Very Common for SRE)

### Question

Write a **shell script** that:

* Checks **CPU usage**
* If CPU usage is **greater than 80%**
* Print **"High CPU usage alert"**

---

## 💡 What the interviewer is testing

They want to see if you know:

* `top` / `mpstat` / `vmstat`
* shell variables
* condition checks (`if`)
* parsing command output

---

# 🧠 Approach

One common way is to use:

```bash
top -bn1
```

* `-b` → batch mode
* `-n1` → run once

This prints CPU statistics.

Example output snippet:

```
%Cpu(s): 85.3 us, 2.0 sy, 0.0 ni, 12.0 id
```

`id` = idle CPU

So:

```
CPU usage = 100 - idle
```

---

# 🧪 Solution Script

```bash
#!/bin/bash

cpu_idle=$(top -bn1 | grep "Cpu(s)" | awk '{print $8}' | cut -d. -f1)

cpu_usage=$((100 - cpu_idle))

if [ $cpu_usage -gt 80 ]
then
    echo "High CPU usage alert: $cpu_usage%"
else
    echo "CPU usage normal: $cpu_usage%"
fi
```

---

# 📌 Example Output

```
High CPU usage alert: 87%
```

or

```
CPU usage normal: 35%
```

---

# ⭐ Follow-Up Questions Interviewers Ask

### ❓ How would you run this every 5 minutes?

Using **cron**

```bash
crontab -e
```

Add:

```
*/5 * * * * /path/cpu_monitor.sh
```

---

### ❓ How would you send an email alert?

Example:

```bash
mail -s "High CPU Alert" admin@company.com
```

---

### ❓ How would you send a Slack alert?

Using **webhook + curl**

```bash
curl -X POST -H 'Content-type: application/json' \
--data '{"text":"High CPU usage detected"}' \
WEBHOOK_URL
```
