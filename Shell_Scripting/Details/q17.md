## ✅ Coding Question #17 — Replace a Word in a File (Config Automation)

### Question

Write a **Linux command/script** to:

* Find a word in a file
* Replace it with another word

Example:

Replace:

```text
ENV=dev
```

with

```text
ENV=prod
```

in file:

```text
config.env
```

---

## 💡 What the interviewer is testing

They want to see if you know:

* `sed`
* configuration automation
* text manipulation

DevOps engineers often **modify config files automatically** during deployments.

---

## 🧪 Simple Solution

```bash
sed -i 's/ENV=dev/ENV=prod/g' config.env
```

---

## 🔎 Explanation

| Part         | Meaning                          |
| ------------ | -------------------------------- |
| `sed`        | Stream editor for modifying text |
| `-i`         | Edit file **in place**           |
| `s/old/new/` | Substitute text                  |
| `g`          | Replace all occurrences          |

---

## Example

### Before

```
ENV=dev
DB_HOST=localhost
```

### After

```
ENV=prod
DB_HOST=localhost
```

---

## ⭐ Replace Only First Occurrence

```bash
sed -i 's/ENV=dev/ENV=prod/' config.env
```

(no `g`).

---

## ⭐ Replace Using Variables (Script Version)

```bash
#!/bin/bash

old="ENV=dev"
new="ENV=prod"
file="config.env"

sed -i "s/$old/$new/g" $file
```

---

## ⭐ Follow-up Questions Interviewers Ask

### ❓ Replace port number in config

```bash
sed -i 's/PORT=8080/PORT=9090/' config.env
```

---

### ❓ Replace text in multiple files

```bash
sed -i 's/dev/prod/g' *.env
```

---

### ❓ Print replaced output without modifying file

```bash
sed 's/dev/prod/g' config.env
```

---

## 🧠 Real DevOps Use Case

`sed` is used heavily in:

* **CI/CD pipelines**
* **config templating**
* **environment variable injection**
* **Kubernetes manifest modification**

Example in pipelines:

```
sed -i "s/IMAGE_TAG/latest/$BUILD_ID/" deployment.yaml
```
