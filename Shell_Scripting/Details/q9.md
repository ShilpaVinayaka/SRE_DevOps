## ✅ Coding Question #9 — Backup a Directory with Timestamp

### Question

Write a **shell script** that:

* Takes a **directory as input**
* Creates a **backup archive**
* Adds a **timestamp** to the backup filename

Example:

Backup `/var/www`

Output file:

```
backup_20260311_163000.tar.gz
```

---

## 💡 What the interviewer is testing

They want to see if you know:

* `tar`
* date formatting
* shell variables
* automation basics

Backups are **very common DevOps automation tasks**.

---

# 🧪 Solution Script

```bash
#!/bin/bash

source_dir="/var/www"

timestamp=$(date +"%Y%m%d_%H%M%S")

backup_file="backup_$timestamp.tar.gz"

tar -czf $backup_file $source_dir

echo "Backup created: $backup_file"
```

---

# 🔎 Explanation

| Command                 | Purpose             |
| ----------------------- | ------------------- |
| `date +"%Y%m%d_%H%M%S"` | Generates timestamp |
| `tar`                   | Creates archive     |
| `-c`                    | Create archive      |
| `-z`                    | Compress with gzip  |
| `-f`                    | Specify filename    |

---

## Example Output

```
Backup created: backup_20260311_163210.tar.gz
```

---

# ⭐ Real Production Version (Better)

DevOps engineers usually store backups in a **backup directory**.

```bash
backup_dir="/backup"

tar -czf $backup_dir/backup_$timestamp.tar.gz $source_dir
```

---

# ⚡ Follow-up Questions Interviewers Ask

### ❓ How to restore this backup?

```
tar -xzf backup_20260311_163000.tar.gz
```

---

### ❓ How to automate daily backup?

Using **cron**

```
0 2 * * * /scripts/backup.sh
```

This runs **every day at 2 AM**.

---

### ❓ How to delete backups older than 7 days?

```
find /backup -type f -mtime +7 -delete
```
